## 功能模块

### 系统角色

#### 管理员

- **药品管理**: 管理药品信息，包括添加、修改药品名称和备注等。
- **科室管理**: 管理科室信息，包括新增、修改科室名称。
- **医生管理**: 管理医生信息，包括修改个人信息和删除离职医生信息。
- **患者管理**: 维护患者注册信息。

#### 医生

- **预约管理**: 审核患者挂号申请。
- **病历管理**: 撰写和记录患者病情。
- **住院管理**: 查看和记录患者住院治疗情况。

#### 患者

- **在线挂号**: 实现在线选择科室和挂号申请。
- **住院信息查看**: 查看个人住院信息。
- **病历查看**: 查看过往病历和当前病情跟踪。
